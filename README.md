# README Generator

A Python tool that generates README.md files and optional MIT LICENSE.

> Note: I don’t have much experience in Python. This project took a lot of effort to make, it doesn’t work perfectly yet, but I will improve it. This project is not forgotten.

## Features

- Fully terminal based
- Customizable readme
- Optional MIT license
- GitHub integrations: license and project link
- Works with Python 3.6+ no external dependencies

## Installation

```bash
git clone https://github.com/8bitDevStudios/readme-gen
cd readme-generator
python readme_generator.py
